# DOTFILES

### HOW TO USE

```sh
$ git clone https://github.com/Umizoko/dotfiles.git

$ cd ~/dotfiles

$ make init
```

### REQUIRED PACKAGE

- git
- tmux
- tmux-plugin
- zsh
- prezto
- fzf
- peco
- vim

# How to build vim

## Required package
- gcc
- libncurses5-dev

## Usage
$ cd vim/src
$ make
$ make install
